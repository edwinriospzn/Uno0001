Regresión Lineal¶

Introduccíon¶

Un problema de regresin lineal comprende el ejercicio de encontrar una relacin entre dos o ms variables considerando un grupo de datos numricos y al poder explicar con una ecuacin el comportamiento de los datos nosotros seriamos capaces de predecir el valor de una variable en funcin de otra con un grado determinado de error.

Historia¶

\ El primer mtodo de regresin lineal registrado en la historia es el de los mnimos cuadrados el cual fue publicado por Legendre en 1805, posteriormente te Gauss public un trabajo en donde desarrollaba de manera ms profunda dicho mtodo. \ \ Linkn: https://es.wikipedia.org/wiki/Regresi%C3%B3n_lineal

Planteamiento del algoritmo¶

En machine learning la regresin lineal es un algoritmo de aprendizaje supervisado que en su versin ms sencilla consiste en trazar una recta que indicara la tendencia de un conjunto de datos continuos. \ \ Linkn: https://www.aprendemachinelearning.com/regresion-lineal-en-espanol-con-python/ \ \ La formula de una recta es: $$Y=mX+b$$ Siendo Y la variable dependiente y X la variable independiente.

#Asignamos X para entrenamiento y las etiquetas de Y
dataX=filtered_data[["Word count"]]
X_train=np.array(dataX)
Y_train=filtered_data['# Shares'].values

#Crear el objeto de regresion lineal
regr= linear_model.LinearRegression()

#Entrenamos el modelo
regr.fit(X_train,Y_train)

#Se hacen las predicciones que en definitiva una linea
Y_pred= regr.predict(X_train)

#Coeficientes obtenidos
print('Coefficients: \n', regr.coef_)
#Valor en donde corta el eje Y en X=0
print('Independent term: \n', regr.intercept_)
#Error cuadrado medio
print('Mean squared error: %.2f' % mean_squared_error(Y_train,Y_pred))
#puntaje de Varianza
print('Variance score: %.2f' % r2_score(Y_train,Y_pred))
